# UNP_SQL_PROJECT_BANKING_DATASET_ANALYSIS

Description:
This project is a MySQL-based Banking Database Management System designed to manage the core operations of a bank. It includes modules for customer details, bank accounts, transactions, loans, and branch information. The system uses relational database concepts with primary keys and foreign keys to maintain data integrity.
Various SQL operations such as CREATE, INSERT, SELECT, UPDATE, DELETE, and ALTER are used to manage the banking data. The project demonstrates real-world banking database design, normalization, and CRUD operations using MySQL. It helps in understanding how banks store, organize, and manage large amounts of financial data efficiently.
